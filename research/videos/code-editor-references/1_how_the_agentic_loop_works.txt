## Summary

- Claude Code is built around an **agentic loop**.  
- The loop has **three phases**: gather context, take action, verify results.  
- Claude begins by **investigating**: running tests, reading errors, searching files, and loading relevant docs.  
- Once it understands the problem, it **acts**: editing files, running commands, generating code.  
- Each action is a **tool call**, and results feed into the next decision.  
- Claude then **verifies**: re‑runs tests or checks outputs to confirm success.  
- If verification fails, it loops back to gather more context and tries again.  
- The loop continues until the task is complete.  
- The **human user remains in control**, able to interrupt, steer, or provide extra context at any point.  
- Models provide reasoning, tools provide execution, and Claude orchestrates the loop as a coding agent.
